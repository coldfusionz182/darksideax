Auth System Information
What is the Software Auth key?
The Software Auth key is a 16‑character token stored in public.users.softwarekey. It is used by certain Darkside tools to verify that you are allowed to run them.[file:94]
How is the key generated?
When you click Gen Key, a random 16‑character string (A‑Z, 0‑9) is created in your browser, then written to your user row as softwarekey via Supabase.[file:94]
Why is there a cooldown?
Keys can be regenerated at most once every 30 seconds to prevent abuse and to keep linked tools stable.
Can normal members use this page?
No. Only users with role admin or owner can generate or view a Software Auth key. Others will see an error message and disabled controls.[file:94]
Can I share my Software Auth key?
Sharing your key is strictly forbidden. It is tied to your Darkside account and may be used to trace abuse back to you. If we detect sharing, the key and account can be revoked.
